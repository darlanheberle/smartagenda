Dúvidas comuns

Antes de começar.

Respostas diretas para você saber exatamente como funciona.

Preciso trocar meu número de WhatsApp?

Não. Você conecta o número profissional que já usa. Seus clientes continuam chamando no mesmo lugar.

Preciso abandonar o Google Agenda?

Não. O Google Agenda continua sendo sua agenda principal. O SmartAgenda apenas consulta os horários e cria os novos compromissos nela.

O cliente precisa instalar outro aplicativo?

Não. Para o cliente, tudo acontece dentro do WhatsApp, como uma conversa normal.

Funciona para qualquer profissão?

Funciona para profissionais que atendem com hora marcada: saúde, beleza, aulas, consultorias, serviços técnicos e muitas outras áreas.

Posso assumir a conversa quando quiser?

Sim. Você pode acompanhar os atendimentos e pausar o assistente pelo painel quando preferir responder pessoalmente.
